Foundations

Color

Warm neutral base, not cold white. One accent (Deep Moss) does all the work. Vermilion is held back for a single alert state — when it appears, it means something.

Typography

Geist Sans for everything human; Geist Mono for what is technical — metadata, tags, numbers, labels. Hierarchy comes from size, weight and space — almost never from colour. Tracking tightens as type grows.

Spacing & Radius

A 4/8 scale gives rhythm; vertical spacing stays on 8px multiples. Radius lives in 16–20px for cards. Nested elements stay concentric: inner radius equals outer minus padding.

Build

Tokens — copy into your build

Drop these straight into the portfolio (Claude Design, Figma variables, or code). One source of truth for every surface.

:root {
  /* base */
  --ink: #111111;
  --charcoal: #1C1C1A;
  --paper: #F7F5F0;
  --ivory: #FBFAF7;
  --stone: #E6E2D8;
  --warm-grey: #8A8781;
  --muted-grey: #C9C5BC;
  /* accent — one tint + one alert */
  --moss: #465242;       /* actions, focus */
  --moss-dark: #3A463A;  /* hover */
  --vermilion: #C84A31;  /* alert only */
  /* radius */
  --radius-sm: 10px;
  --radius-md: 16px;
  --radius-lg: 20px;
  --radius-full: 999px;
  /* type */
  --font-sans: 'Geist', system-ui, sans-serif;
  --font-mono: 'Geist Mono', ui-monospace, monospace;
  /* motion */
  --ease: cubic-bezier(0.22, 0.61, 0.36, 1);
  --dur: 180ms;
}
