Status — neutral by default; the only place Vermilion is allowed, and only for a genuine exception.

```jsx
<Status>In progress</Status>
<Status variant="done">Resolved</Status>
<Status variant="urgent">Urgent incident</Status>
```

Mono pill with a leading indicator: a muted dot (default), a ✓ (done), or a Vermilion dot + wash (urgent). Never use `urgent` decoratively — its scarcity is what gives it meaning.
